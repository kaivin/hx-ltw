# 长尾词管理系统

本项目主要用于创建长尾词页面，通过组合不同模块，创建出不同的页面，目的在于提高出长尾词站的速度，以及提高代码复用率

# 实现内容

* 区分首页、关于我们、联系我们、产品列表、详情、案例列表、详情、长尾词列表
* 区分页面框架类型
* 区分页面模块类型
* 可预览模块，组合页面样式、可复制模块、页面代码、样式
* 实现随机抽取数据组合页面

# 命令

1. 安装

```
yarn
```
2. 运行开发环境

```
yarn start
```

3. 输出测试环境

```
yarn test
```

4. 输出生产环境

```
yarn build
```
